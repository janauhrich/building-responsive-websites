One Grid to Rule Them All

We have 12 columns to work with on all four screen sizes. What we control is when a column goes from laying horizontal to stacking vertically. By giving something a col of 1 you are saying "If the screen is bigger than this then let me take up 1 column but if it's smaller then stack 'em."
